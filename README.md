# Vector DB QA System — RAG Pipeline

A Retrieval-Augmented Generation (RAG) based QA system using 
ChromaDB and Pinecone with free HuggingFace embeddings.

## Tech Stack
LangChain, ChromaDB, Pinecone, HuggingFace (all-MiniLM-L6-v2), Groq API, Python

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. Add your Groq API key in the notebook
4. Run vector_db_qa.ipynb
